# Your Notes Web App

### HTML | CSS | JavaScript

![Project Image](/static/img/markdown_image.png)

This is a web app that runs on the client's browser. The user can:

1. create an account
2. create notes
3. edit notes
4. delete notes or mark them as complete
5. update their user profiles and more
